# ProyectoFinal-Fidness
## Descripción
Sistema desarrollado en Java utilizando NetBeans y Swing para la administración de un gimnasio.

## Funcionalidades implementadas (Avance 2)

- Inicio de sesión
- Gestión de usuarios
- Catálogo de ejercicios
- Creación de rutinas
- Persistencia mediante serialización
- Interfaz gráfica

## Tecnologías utilizadas

- Java
- Swing
- Programación Orientada a Objetos
- Serialización
- Git y GitHub
